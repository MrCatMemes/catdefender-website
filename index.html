So funktioniert’s

  1. Programm starten → „Scan starten“ klicken.
  2. Ordner auswählen → CatDefender prüft Dateien.
  3. Verdächtige Dateien → Quarantäne.
  4. Von dort löschen oder prüfen.

FAQ

Ist CatDefender kostenlos?

Ja. CatDefender ist ein Hobby-/Community-Projekt und wird 100 % kostenlos bereitgestellt.

Hat CatDefender Werbung oder versteckte Kosten?

Nein. Es gibt keine Werbung, keine Abos und keine versteckten Kosten.

Kann CatDefender alle Viren finden?

Nein. Es ist ein leichtes Tool für Basis-Scans. Es ergänzt Microsoft Defender, ersetzt ihn aber nicht.

Sammelt CatDefender Daten?

Nein. Alles läuft lokal, es gibt kein Tracking und keine Cloud-Uploads.

Wo landen Dateien aus der Quarantäne?

Im Ordner C:\CatDefender\Quarantine. Dort können sie geprüft oder gelöscht werden.

Ist CatDefender sicher oder ein Fake-Antivirus?

CatDefender ist ein transparentes Projekt. Keine versteckten Funktionen, keine Malware, kein Scam.

Wird CatDefender weiterentwickelt?

Ja! Geplant sind Log-System, automatische Updates und eine erweiterte UI.
