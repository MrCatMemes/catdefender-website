So funktioniert’s

  1. Start → „Scan starten“.
  2. Ordner wählen → Dateien werden geprüft.
  3. Funde landen in C:\CatDefender\Quarantine.
  4. Im Quarantäne-Fenster prüfen oder löschen.

Hinweis: schlankes Tool für Basis-Checks, kein vollwertiger Enterprise-Ersatz.

FAQ

Ist CatDefender kostenlos?

Ja. 100 % kostenloses Hobby-/Community-Projekt.

Hat CatDefender Werbung oder versteckte Kosten?

Nein. Keine Werbung, keine Abos, kein Tracking.

Ersetzt CatDefender Microsoft Defender?

Nein, es ergänzt ihn nur. Defender bleibt deine Haupt-Protection.

Sammelt CatDefender Daten?

Nein. Alles passiert lokal, keine Cloud-Uploads.

Wo ist die Quarantäne?

C:\CatDefender\Quarantine. Von dort prüfen oder löschen.

Wird weiterentwickelt?

Jep: Logs, Auto-Updates, erweiterte Signaturen stehen auf der Liste.
